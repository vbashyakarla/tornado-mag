Tornado Magnitude Prediction

Project Overview

This project contains an exercise in modeling tornado severity. The analysis was performed in R and inspired by Tidy Tuesday.

Tornadoes were a regular occurence during my childhood in Missouri and Kansas, so I couldn't pass up an opportunity to analyze tornado-related data. As the repo's README states, this data was originally compiled from the Severe Weather Maps, Graphics, and Data page of the National Oceanic and Atmospheric Administration's National Weather Service Storm Prediction Center. The data document measurements associated with 68,693 tornadoes from 1950 to 2022.

The analysis includes some exploratory data analysis, feature engineering, and modeling. Tornado magnitudes are predicted using:

  • XGBoost (with a number of tuned hyperparameters)
  • Standard Poisson Regression
  • Zero-Inflated Poisson Regression
